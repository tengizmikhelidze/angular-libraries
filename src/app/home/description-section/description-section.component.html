გზამკვლევი ანგულარის სამყაროში

ქართული პლათფორმა ანგულარის სამყაროში. გვსურს ერთ სივრცეში გავაერთიანოთ ყველა ანგულარის სერვისები რათა გავუმარტივოთ დეველოპერებს ამ დიდ ხელსაწყოსთან მუშაობა. გთოხვთ გაითვალისწინოთ რომ საიტი და სერვისები დეველოპმენტის პროცესშია და კონტრიბუციის სურვილის შემთხვევაში დაგვიკავშირდეთ
